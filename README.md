# 📊 SQL Data Analysis Project

##  Overview
This project demonstrates SQL queries for managing and analyzing an online store database.

## Technologies Used
- SQL (MySQL)

##  Project Structure
- schema.sql → Database & table creation
- queries.sql → Data analysis queries

##  Features
- Create database and tables
- Insert sample data
- Perform filtering and search queries

##  How to Run
1. Run schema.sql
2. Run queries.sql

##  Use Cases
- Product filtering

- Data insights extraction
